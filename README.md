# Grocery Management System

A simple **menu-driven grocery inventory management system** implemented in Java using **parallel arrays**.  
This project was developed collaboratively for CS3354 to practice Java programming and GitHub collaboration.


## Project Description

This program manages grocery items using three parallel arrays.

## Features

- View inventory items
- Restock existing items
- Menu-driven interface
- Parallel array data management
- Javadoc documentation
- Git branch collaboration workflow

---

## How It Works

The program displays a menu with three options:

1. View Inventory — Displays all available grocery items.
2. Restock Item — Adds stock to an existing item.
3. Exit — Terminates the program.
